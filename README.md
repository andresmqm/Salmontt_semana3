En este proyecto tiene como finalidad aplicar los fundamentos de la Programacion orientado a objetos (POO)
El principal objetivo es la gestion basica de la informacion de los empleados 
En el proyecto se desarrolla varias clases como (Direccion, Empleados y Personas) cada una con su constructor y en el caso de la clase empleado un constructor que valide el Rut de manera correcta
En el main se crea distintas instancias donde se prueba los resultados en consola
En sintesis con este proyecto busco reforzar el uso de constructores, expresiones regulares de validacion de datos y encapsulamiento
